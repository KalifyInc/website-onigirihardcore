A Microsoft atualizou sua página oficial de suporte para informar usuários sobre o fim do ciclo de vida do Windows 10, que deixará de receber atualizações de segurança em outubro de 2025. O site da empresa agora fornece informações, ferramentas e dicas para aqueles que estão usando o sistema operacional de 2015.

A página — que antes listava apenas o Windows 7 e Windows 8.1 (descontinuados em 2020 e 2023, respectivamente) — passou a lembrar os usuários sobre o fim de suporte do Windows 10, alertando que essa versão deixará de receber atualizações de segurança e suporte técnico em 14 de outubro de 2025.

![](https://t2.tudocdn.net/712972?w=660&h=424)

Como recomendação, a empresa indica que os usuários atualizem seus computadores para o Windows 11 — isto é, caso sejam compatíveis com os requisitos mínimos da atual versão do sistema operacional. A página destaca os novos recursos do Windows 11, além de exibir um comparativo de recursos com o Windows 10.

Para aqueles que não possuem um PC compatível com Windows 11, a Microsoft exibe um guia para comprar um novo desktop ou notebook que sai da fábrica já executando a versão mais recente do sistema operacional. Além disso, a página mostra como fazer a transferência de arquivos do computador antigo para um novo dispositivo.

A versão do sistema operacional mais usada no mundo ainda é o Windows 10, com 67,23%, enquanto seu sucessor ainda detém somente 28,18%. Curiosamente, em março, a geração mais antiga do software observou um ligeiro crescimento na parcela de usuários, denotando uma baixa preferência pelo Windows 11.

É importante ressaltar que existem riscos em utilizar um sistema operacional que perdeu o suporte da empresa. Sem atualizações de segurança, o Windows 10 pode não saber lidar com novos tipos de malwares, deixando o computador exposto a invasões.

![](https://t2.tudocdn.net/619582?w=660&h=371)

Para aqueles que precisam de mais tempo para migrar para o Windows 11, a Microsoft oferecerá o Programa de Segurança Estendida que, pela primeira vez, poderá ser adquirido por usuários domésticos — e não somente por empresas.

Embora a Microsoft já tenha revelado que os clientes corporativos poderão desembolsar US$ 61/ano (cerca de R$ 315/ano) por PC para receber atualizações de segurança após a data de encerramento de suporte do Windows 10, ainda não há informações sobre o valor que será cobrado dos usuários domésticos.
