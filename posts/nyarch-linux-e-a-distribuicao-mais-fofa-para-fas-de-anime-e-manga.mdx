NyArch Linux é um sistema operacional feito especialmente para aqueles que amam a cultura japonesa, especialmente a cultura pop... mais especificamente, mangá e anime. Eu, particularmente, passei muitos anos assistindo anime e lendo mangá e, embora tenha se passado um tempo, ainda mergulho de vez em quando e assisto a um anime aleatório para me reconectar com as memórias da adolescência ou simplesmente para desfrutar de uma animação e narrativa muito boas.

## Principais pontos do ZDNET
- NyArch Linux é um excesso de fofura na forma de um sistema operacional de código aberto e está disponível agora gratuitamente.
- NyArch Linux oferece muitas personalizações para aumentar seu kawaii, bem como uma gama completa de aplicativos GNOME.
- Não é a distribuição Linux mais estável que já usada.

Mas ter uma distribuição Linux inspirada em anime/mangá? Isso soa divertido. E é!

No seu cerne, NyArch Linux é Arch Linux com a área de trabalho GNOME. Isso, combinado com sua facilidade de instalação, deve atrair muitos. Mas, em seguida, você adiciona toda a fofura kawaii/uwu e a distribuição tem muito a oferecer.

Só não leve isso muito a sério.

Embora em casa ou na escola, NyArch seja muito divertido, provavelmente não é a distribuição Linux que você usaria no escritório. Mas ambientes empresariais não são o público-alvo do NyArch. Isso é tudo sobre excesso de fofura e um sistema operacional que lembra você de aproveitar seu lado amante de animes e ansioso por fofura.

Isso não significa que NyArch seja algo para ser ridicularizado ou zombado. É Linux, o que significa que pode ser e fazer qualquer coisa. Além disso, é Arch Linux, então é muito sólido, seguro e personalizável.

NyArch também tem alguns truques divertidos na manga, como a capacidade de escolher entre seis layouts de área de trabalho diferentes (painel inferior, GNOME padrão, painel superior com menu, painel superior com menu e dock, painel superior com dock, painel superior com dock à esquerda) e uma ferramenta de temas que inclui animações como duas das minhas favoritas, Janelas Ondulantes e Cubo da Área de Trabalho.

![](https://www.zdnet.com/a/img/resize/698005dc216ffe6e5a7f5125b0fbbacfb4400846/2024/05/20/bbb5370d-6765-470c-aaea-627e3595a6bc/nyarch2.jpg?auto=webp&width=1280)

Uma personalização que deve ser destacada é o Material UwU. Esta é uma extensão GNOME que usa o papel de parede escolhido para automaticamente tematizar seu sistema. É muito parecido com o que o Google fez com o Android e Material You e funciona muito bem. Selecione uma nova imagem de papel de parede e o tema ajusta-se de acordo. Existem até dois aplicativos que baixarão uma imagem inspirada em anime/mangá aleatória (baseada em personagens waifu e kemonomimi) para seu papel de parede.

A lista de aplicativos padrão inclui Firefox, os aplicativos GNOME padrão (Arquivos, Clima, Vídeos, Calculadora, Terminal, Editor de Texto, etc.), Geary, Lillypop (player de música), Chromium, Gerenciador de Extensões, Flatseal (gerenciador de permissões de flatpak).

Também há Komikku, um aplicativo que facilita o download e a leitura de vários títulos de mangá de diversas fontes.

![](https://www.zdnet.com/a/img/resize/938d000c0f5081ce87c3c3f8492c9b37ef3fc167/2024/05/20/7f2fcff5-9544-42c2-93e6-553883451bb8/nyarch3.jpg?auto=webp&width=1280)

O único aplicativo ausente no NyArch é um pacote de escritório, mas você pode instalar o LibreOffice a partir do Package Apps.

Eu diria que nem tudo foi totalmente umai com o NyArch. Tive vários aplicativos que travaram (como Pacotes), o que tornou difícil trabalhar com ele. Por causa disso, acabei tendo que instalar aplicativos pela linha de comando (como `sudo pacman -Syu libreoffice`). Eu estava disposto a relevar isso, considerando que o NyArch não é desenvolvido por uma equipe de desenvolvedores e porque sei como lidar com suas deficiências. No entanto, para aqueles que podem não ter as habilidades para lidar com uma distribuição Linux que pode não funcionar conforme as expectativas oferecidas pelo sistema operacional em que foi baseado, o NyArch pode ser um pouco desafiador.

Lembre-se, no entanto, que testei o NyArch usando uma máquina virtual VirtualBox. É bem possível que os problemas que experimentei tenham sido causados por a distribuição não estar rodando em uma máquina própria (conhecida como "bare metal").

Se você está preocupado com esses problemas, sempre há o NyArcher, que é um script de shell para instalar as customizações do NyArch. Você pode usar o NyArcher nas distribuições Arch, Fedora e até mesmo Ubuntu.

## NyArcher funciona assim:

### Arch Linux

<blockquote>
sudo pacman -S curl python3-pip flatpak svn gnome-menus kitty wget git neofetch npm nodejs btop gnome-menus gnome-shell-extensions
sudo pacman -S python-pywal
</blockquote>

### Fedora

<blockquote>
sudo dnf install curl flatpak python3-pip svn gnome-menus kitty wget git neofetch npm nodejs btop gnome-menus gnome-extensions-app
sudo pip3 install pywal
sudo cp /usr/local/bin/wal /usr/bin/wal
</blockquote>

### Ubuntu

<blockquote>
sudo apt install curl python3-pip flatpak subversion gnome-menus kitty wget git neofetch npm nodejs btop gnome-menus gnome-shell-extension-prefs
sudo pip3 install pywal
</blockquote>

Entenda que, ao usar o script, você não obterá a experiência completa do NyArch.

Do meu ponto de vista, NyArch é muito divertido de experimentar. Eu usaria como minha distribuição principal? Não. Vale a pena tentar? Certamente. Se você está procurando por um excesso de fofura, experimente esta abordagem no Arch Linux e veja se não faz você sorrir aleatoriamente enquanto passa o dia.
